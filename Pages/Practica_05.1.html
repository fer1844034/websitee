Aqui veremos el uso de las 10 etiquetas utilizadas HTML y sus parametros.

Etiqueta button.

La etiqueta de HTML button representa un elemento cliqueable de tipo botón que puede ser utilizado en formularios o en cualquier parte de la página que necesite un botón estándar y simple de aplicar. De forma predeterminada, los botones HTML se presentan con un estilo similar en todas las plataformas, estos estilos se pueden cambiar utilizando CSS. Sus etiquetas son: (button) y (/button) ambas obligatorias para el funcionamiento de la etiqueta.


Etiqueta Nav.

El elemento HTML nav representa una sección de una página cuyo propósito es proporcionar enlaces de navegación, ya sea dentro del documento actual o a otros documentos. Ejemplos comunes de secciones de navegación son menús, tablas de contenido e índices. Sus etiquetas son (nav) y (/nav) ademas de implementar etiquetas ya conocidas como (ul) que es la que nos sirve para crear una lista desordenada, tambien se puede usar la etiqueta (ol) que se usa para crear listas ordenadas y asimismo la etiqueta (li) para agregar un elemento a la lista y por ende (/li) para cerrar dicha etiqueta.


Etiqueta Aside.

El elemento HTML aside representa una sección de una página que consiste en contenido que está indirectamente relacionado con el contenido principal del documento. Estas secciones son a menudo representadas como barras laterales o como inserciones Se deben de usar las 2 etiquetas para su funcionamiento (aside) y (/aside) y se puede insertar adentro del aside cualquier elemento que acepte contenido dinámico. Ten en cuenta que un elemento aside no debe ser un descendiente de un elemento address.


Etiqueta Figure

El elemento HTML figure representa contenido independiente, a menudo con un título. Si bien se relaciona con el flujo principal, su posición es independiente de éste. Por lo general, se trata de una imagen, una ilustración, un diagrama, un fragmento de código, o un esquema al que se hace referencia en el texto principal Esta etiqueta requiere la etiqueta de entrada (figure) y la etiqueta de salida (/figure).




Etiqueta Figcaption

El elemento HTML figcaption representa un subtítulo o leyenda asociado al contenido del elemento padre (figure), pudiendo ser colocado como primer o último hijo. Es importante destacar que el elemento figcaption es opcional. Requiere de un elemento superior como lo es el caso de la etiqueta figure y la etiqueta de entrada(figcaption y de cierre(/figcaption).


Etiqueta Header

El elemento de HTML Header representa un grupo de ayudas introductorias o de navegación. Puede contener algunos elementos de encabezado, así como también un logo, un formulario de búsqueda, un nombre de autor y otros componentes. Van de H1 a H6 respecto a su importancia o relevancia. Cualquier contenido dinámico es admitido, pero sin otro o footer descendente de la misma.


Etiqueta q.

El elemento q (cita en español) crea citas en linea, marca las citas a otros autores o documentos. Se deberan de poner las etiquetas de entrada (q) y de salida (/q) para marcar cuando se acabo de citar a alguien.


Etiqueta Blockquote

La etiqueta blockquote (cita en bloque en español) sirve para crear citas en bloque, marca las citas a otros autores o documentos, se puede decir que es el hermano mayor de q por tener mas funciones. Como en las demas etiquetas se pide la etiqueta de entrada (blockquote) y la etiqueta de salida (/blockquote)


Etiqueta audio.

El elemento audio se usa para insertar contenido de audio en un documento HTML o XHTML. El elemento audio se agregó como parte de HTML 5. Etiqueta especial, que contiene bien un atributo src que te linke hacia una direccion en tu memoria, ademas de uno o más elementos source, seguido de contenido dinámico o contenido estático , ademas de una etiqueta llamada controls para habilitar el play, pause etc.


Etiqueta video.

El elemento video se utiliza para incrustar vídeos en un documento HTML o XHTML.srcque en otras palabras es la URL del vídeo que se va a insertar. Es opcional; podrás optar, en su lugar, por el elemento source dentro del bloque de vídeo para especificar el video que se va a incrustar. width La anchura del área de visualización del vídeo en píxeles CSS. Como ya se explico arriba se usan dentro de esta etiqueta otras etiquetas que son src o source y width para elegir el tamaño del video en la pagina web.
